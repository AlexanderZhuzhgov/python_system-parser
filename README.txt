Для работы проэкта необходимы такие библиотеки как 
1.Библиотека Selenium
Для ее установки в терминале нужно ввести команду sudo pip3 install selenium
2.Библиотека tkinter
Данная библиотека предустановлена в python, но может быть у вас она остутсвует
Для ее установки необходима следующая команда sudo apt-get install python3-tk
Более никаких библиотек для работы скрипта не потребуется.

Так же нужно переместить драйвер <chromedriver> в папку /usr/local/bin/
Осуществить это можно с помощью команды sudo mv /нынешний путь к файлу/ /usr/local/bin/
Если вы экспортировали архив в /home/ то команда будет такой sudo mv chromedriver /usr/local/bin/
Делается это для работы библиотеки selenium

Пункты запуска программы:
1.Первым из двух файлов мы должны запустить скрипт info.py
Он отвечает за сбор информации о системе и запись её в файл sysinfo
Для запуска данного скрипта необходимо прописать команду в терминале sudo python3 /путь к файлу/info.py
Если вы экспортировали архив в домашнюю папку(home) то команда будет такой sudo python3 info.py
ВАЖНО первый скрипт мы должны запускать с правами администратора(то есть sudo)
Это необходимо для того что бы вывести полную информацию о системе с помощью чтения файла lshw
Без прав администратора не все строки файла доступны.
После запуска, скрипт соберет информацию и откроется графический интерфейс с выбором языка вашей системы.
Вам необходимо нажать на кнопку с языком вашей системы(НЕ РАСКЛАДКИ, а именно системы)
После нажатия на кнопку скрипт закроется, и запишет информацию о системе в файл sysinfo

2.Вторым файлом мы запускаем файл parseinfo.py
Он отвечает за чтение информации из файла sysinfo, и парсинг сайтов по данным из этого же файла.
Для запуска данного скрипта необходимо прописать команду в терминале python3 /путь к файлу/parseinfo.py
Если вы экспортировали архив в домашнюю папку(home) то команда будет такой python3 info.py

ВАЖНО второй скрипт мы должны запускать без прав администратора(тоесть без sudo)
Selenium не дает адекватно парсить сайты если запустить скрипт с правами администратора
Тоесть первый скрипт мы запускаем с правами администратора(который отвечает за сбор системной информации)
второй скрипт мы запускаем БЕЗ прав администратора(который отвечает за парсинг по собранной системной информации)
После введения команды произойдет парсинг сайтов
Данный процесс будет занимать до 5 минут если у вас быстрый интернет
К сожалению никак ускорить это нельзя, так как данные сайты используют защиту от парсинга
По этому требуется использовать библиотеку selenium, которая иммитирует некоторые действия пользователя
После парсинга всех сайтов, откроется окно с выведенной информацией о стоимости ваших комплектующих


Так же ВАЖНО:
В файле info.py в самом конце кода укажите где находится ваш файл,если вы распакуете архив в домашнюю папку ,не стоит менять ничегосама строка будет выглядетьтак filetext = open('sysinfo', 'w')
С вашим путем будет выглядеть так
filetext = open('/ваш путь к файлу sysinfo/', 'w')
Соответственно во 2ом файле parseinfo.py небходимо так же указать путь к sysinfo
Если вы распаковали арихив в домашнюю папку(home)
То строчку не нужно менять и она будет выглядеть так information = open('sysinfo','r')
С вашим измененым путем будет выглядеть так information = open('/ваш путь к sysinfo/sysinfo','r')
